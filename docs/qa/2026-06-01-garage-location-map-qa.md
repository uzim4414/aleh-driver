## מה שנבנה

מפת Google Maps אינטראקטיבית מוטמעת בתוך כרטיס בקשת מוסך בפלייט מנג'ר, מציגה את מיקום הרכב ברגע שהנהג פתח את הבקשה.

### מה השתנה

| קובץ | שינוי |
|------|-------|
| `code.js` | `getGarageRequests` חושף `lat`/`lng` ב-response |
| `index.html` | `_acGarageRequestCard` מציג iframe + label + קישור |

### מיקום בכרטיס
```
תיאור התקלה
─────────────────────────────
📍 מיקום בשעת הפתיחה        ← חדש
[מפה אינטראקטיבית 160px]    ← חדש
פתח במפה ↗                  ← חדש
─────────────────────────────
רייכמן, סוכנות מורשית טויוטה
...
```

### טכנולוגיה
- **Google Maps Embed URL** — `https://maps.google.com/maps?q={lat},{lng}&z=16&output=embed`
- **אפס API key** — embed URL ציבורי, חינמי
- **iframe** עם `sandbox="allow-scripts allow-same-origin allow-popups"`, `loading="lazy"`
- **מוסתר** כשאין קורדינטות (lat=0 או lng=0)

## הערות

- `lat`/`lng` נשמרים כבר מהיום הראשון ב-`FIELD_EVENTS` — כל הבקשות הישנות שנפתחו מהאפליקציה כוללות קורדינטות
- בקשות ישנות שנפתחו **לפני** עדכון האפליקציה (כשלא היה GPS) — לא תוצג מפה (lat=0)
- מפה נפתחת בטעינה עצלה (`loading="lazy"`) — לא מאטה את טעינת הכרטיס
